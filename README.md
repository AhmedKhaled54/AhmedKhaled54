### 🛒 E-Commerce API 
> Scalable backend API handling product catalog, cart, orders, payments, and user authentication.

- Architected solution using **Clean Architecture + CQRS** pattern, separating read/write responsibilities for maintainability and scalability
- Applied **Specification Pattern** to build reusable, composable query filters — reducing code duplication across data access layers
- Implemented secure **JWT Authentication** and **Role-Based Authorization (RBAC)** using ASP.NET Identity
- Integrated **Stripe** payment gateway for order checkout and **SMTP** for transactional email notifications
- Built full API suite covering auth, product catalog, shopping cart, and order management
